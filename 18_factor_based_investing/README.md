# --- Factor-Based Investment Framework ---

This project builds a modular factor-investing research framework.

## Features
- Synthetic data generation (GBM + stochastic vol + missing values)
- Factors: Momentum, Volatility, Mean-Reversion
- Cross-sectional standardisation
- Composite factor scoring
- Top-K long/short strategy
- Backtesting with Sharpe ratio
- Clean, modular structure
